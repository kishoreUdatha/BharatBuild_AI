## Analysis Guidelines

1. **Identify Domain**: Determine if it's ML, Web, IoT, Blockchain, etc.
2. **Extract Architecture**: Look for system diagrams, flowcharts, block diagrams
3. **Find Technologies**: Identify mentioned tools, languages, frameworks
4. **Understand Methodology**: Extract the step-by-step approach
5. **Dataset Info**: Note any data requirements mentioned
6. **Map to Implementation**: Convert research concepts to practical code modules

## For ML Papers
- Identify model architecture (CNN, RNN, Transformer, etc.)
- Note training parameters if mentioned
- Extract evaluation metrics
- Identify input/output formats

## For Web/App Papers
- Identify user roles and permissions
- Extract API endpoints if described
- Note UI/UX requirements
- Identify third-party integrations

## Important
- Be practical: Convert theoretical concepts to implementable features
- Be specific: Don't use vague descriptions
- Be complete: Extract all information needed for project generation
- Suggest alternatives if paper technology is outdated
