What is a Socket?

A socket is an endpoint for communication between two machines. It is a combination of IP address and port number. Sockets allow applications to communicate over a network using standard protocols like TCP and UDP.

üîë Key Concepts

  • ‚Ä¢ Socket = IP Address + Port Number
  • ‚Ä¢ Server: Creates socket ‚Üí binds ‚Üí listens ‚Üí accepts
  • ‚Ä¢ Client: Creates socket ‚Üí connects
  • ‚Ä¢ Types: SOCK_STREAM (TCP), SOCK_DGRAM (UDP)

Socket Creation in Python:

import socket

# Create a TCP socket
server_socket = socket.socket(
    socket.AF_INET,
    socket.SOCK_STREAM
)

# Bind to address
server_socket.bind(('localhost', 8080))

# Listen for connections
server_socket.listen(5)

Problem Description

TCP Echo Server

Write a TCP server that echoes back any message received from the client.

Requirements:

  1. Listen on port 9999
  2. Accept client connections
  3. Receive message from client
  4. Send back the same message
  5. Close connection

Example:

Input: "Hello"

Output: "Hello"

Constraints:

  • Port: 9999
  • Single client at a time
  • Max message: 1024 bytes

üí° Hints:

  • Use socket.bind() to bind address
  • Use socket.accept() for connections
  • Use conn.recv() to receive data
